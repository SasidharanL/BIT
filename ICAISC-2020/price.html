PROCEDURE FOR PAPER SUBMISSION

  • Submit your paper via Email - icaisc20@gmail.com
  • Registration form and other details will be sent only if paper is accepted

  • Paper is to be prepared using the following guidelines

  • Paper Format:Standard, Single column format limited to 8 pages only
  • Paper Size:A4
  • Font: Times New Roman (All contents of the entire paper is to be typed using Times New Roman Font)
  • Paper Title:Centre and bold with font size 14 points
  • Paper Contents: font size 12 points with sub-title bold
  • Authors Name(s), Affiliation(s): Center with font size 12 points
  • Table Captions should be placed above the table with font size 12 points
  • Figure Captions should be placed below the figure with font size 12 points
  • Reference should be numbered starting from 1 with font size 10 points
  • All subtitles and contents should be left aligned and justified
  • Avoid using text boxes and make all the tables and figures in line with text
  • Set Line Spacing to Single and Set paragraph spacing to 0 (for all contents and subtitles)
  • Manuscripts should be a minimum of 5 (five) single-sided pages (suggested maximum of 10 pages)
